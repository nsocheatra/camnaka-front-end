# CAM NAKA CO., LTD. Website

A responsive, modern website for CAM NAKA CO., LTD., specializing in import and export of mother and baby products.

## About

This website showcases the company's services, products, and contact information. Built with Next.js, TypeScript, and Tailwind CSS for optimal performance and SEO.

## Features

- **Homepage**: Hero section with company branding and product highlights
- **About Page**: Company history, mission, values, and vision
- **Products Page**: Showcase of product categories with descriptions
- **Contact Page**: Contact form, company details, and embedded Google Map
- **Responsive Design**: Mobile-friendly layout with clean, modern styling
- **SEO Optimized**: Fast loading and search engine friendly

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies Used

- **Next.js** - React framework for production
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **ESLint** - Code linting
